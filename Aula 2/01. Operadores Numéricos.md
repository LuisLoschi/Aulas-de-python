# Operadores Numéricos

Operadores           | Valor de exemplo
-------------------- | ------
Soma ( + )           | 5 + 2 = 7
Subtração ( - )      | 5 - 2 = 3
Multiplicação ( * )  | 2 * 2 = 4
Divisão ( / )        | 5 / 2 = 2,5
Resto ( % )          | 5 % 2 = 1
Divisão inteira( // )| 5 // 2 = 2
Expoente ( ** )      | 5 ** 2 = 25
