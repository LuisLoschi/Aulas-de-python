# Arquivos ‚Äì readlines()

Retorna uma lista com todas as linhas do nosso arquivo.

~~~python
arq = open("teste.txt", "w")
arq.write("Nova Linha\n")
arq.write("Nova Linha\n")
arq.close()

arq = open("teste.txt", "r")
texto = arq.readlines()
arq.close()

print(texto)

#['Nova Linha\n', 'Nova Linha\n']
~~~
